# 수푸름 설치하기

## 1. 설치 파일 받기

**아래 링크를 누르면 설치 파일이 바로 받아져요.**

### 👉 [여기를 누르면 바로 받아져요 — 수푸름 설치 파일 내려받기](https://github.com/hacker943410-debug/supureung-releases/releases/download/v0.0.1/supureung-0.0.1-setup.exe) (현재 최신: v0.0.1)

누르면 `supureung-0.0.1-setup.exe` 파일이 컴퓨터에 저장됩니다. (파일 크기는 약 80MB이며, 인터넷 속도에 따라 잠시 기다리실 수 있어요.)

## 2. 받은 파일 실행하기

1. 다운로드가 끝나면 화면 아래(또는 다운로드 폴더)에서 `supureung-0.0.1-setup.exe` 파일을 찾아 **두 번 클릭**하세요.
2. 설치 화면의 안내를 따라가면 자동으로 설치가 진행됩니다.
3. 설치가 끝나면 바탕화면이나 시작 메뉴에서 수푸름을 실행할 수 있어요.

## 3. 이런 창이 뜨면 이렇게 하세요

### Windows가 파란 경고 창(알 수 없는 게시자)을 띄울 때

아직 정식 인증서가 없어 Windows가 "Windows에서 PC를 보호했습니다" 같은 파란 경고 창을 띄울 수 있어요. 악성 프로그램이 아니니 아래처럼 진행하세요.

1. 창 안의 **"추가 정보"** 글자를 누르세요.
2. 그러면 **"실행"** 버튼이 나타납니다. 그 버튼을 누르면 설치가 계속됩니다.

### 브라우저가 "안전하지 않음"이라며 다운로드를 막을 때

브라우저(엣지·크롬 등)가 처음 보는 파일이라 "이 파일은 일반적으로 다운로드하지 않는 파일입니다" 같은 경고를 보여줄 수 있어요.

1. 경고 옆이나 아래의 **"유지"**(또는 "계속 유지", "다운로드 유지") 버튼을 누르세요.
2. 그러면 다운로드가 마저 완료됩니다.

## 4. 참고

- 이 페이지(Releases)에는 낯선 이름의 파일도 함께 있어요.
  - `latest.yml`·`.blockmap` 파일은 **자동 업데이트를 위한 부속 파일**이라 직접 받으실 필요가 없습니다.
  - `portable.zip`은 **설치 없이 압축을 풀어 바로 실행하는 전문가용 판**입니다. 일반적으로는 필요 없습니다.
  - 즉, **`supureung-0.0.1-setup.exe`(설치 파일) 하나만 받으시면 됩니다.**
